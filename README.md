# Week 6: Asynchronous JavaScript & APIs

## Project Overview

This project focuses on mastering asynchronous JavaScript concepts including:

- Callbacks
- Promises
- Async/Await
- Fetch API
- API Integration
- DOM Manipulation
- Error Handling
- Local Storage

The main deliverable for this week is a fully functional **Weather Dashboard** built using the OpenWeatherMap API.

# Mini Project: Weather Dashboard

## Features

### Weather Search
- Search weather by city name
- Fetch live weather data from OpenWeatherMap API

### Weather Information Display
- City name
- Temperature
- Weather description
- Weather icon
- Humidity
- Wind speed
- Pressure
- Feels-like temperature

### Error Handling
- Invalid city handling
- Network error handling
- Loading states

### Recent Searches
- Saves last 5 searches
- Clickable search history
- Data stored using localStorage

### Bonus Features
- Temperature unit toggle
- 5-day forecast
- Current location weather
- Dynamic backgrounds based on weather conditions
